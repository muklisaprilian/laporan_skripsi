<ul>
<li>Codeigniter</li>
<li>Bootstrap Admin LTE</li>
<li>Javascript</li>

Cara menggunakan : 
<ul>

<li>Ganti settingan database, dan sesuaikan dengan settingan punya kalian</li>
</ul>
